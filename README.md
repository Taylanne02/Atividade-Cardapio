# 🍔 Atividade - Cardápio

Projeto desenvolvido utilizando **React + Vite** para a criação de um cardápio simples de lanchonete.

## 🛠️ Passo 1

✅ Criação do projeto com React + Vite  
✅ Estrutura inicial do `App.jsx`  
✅ Criação dos itens do cardápio  
✅ Exibição de nome e preço dos produtos  

## 🛠️ Passo 2

✅ Criação do componente `ItemCardapio.jsx`  
✅ Separação dos itens em componentes reutilizáveis  
✅ Uso de `props` para receber nome e preço  
✅ Renderização dos 5 itens utilizando o componente `<ItemCardapio />`
